# 🎨 Chromatic AI

> An Interactive Educational Platform for Constraint Satisfaction Problems and Graph Algorithm Visualization

Chromatic AI is a web-based educational platform designed to simplify the understanding of **Constraint Satisfaction Problems (CSPs)** and **Graph Theory algorithms** through real-time visualizations and interactive simulations. The project bridges the gap between theoretical concepts and practical applications by allowing users to observe how algorithms make decisions step by step.

---

## 🚀 Features

### 🗺️ Interactive Map Coloring
- Visualizes the Graph Coloring Problem using real-world datasets.
- Supports coloring of the **India Map** and other graph models.
- Ensures adjacent regions do not share the same color.

### 🧠 Constraint Satisfaction Algorithms
Implemented CSP algorithms include:
- Backtracking
- Minimum Remaining Values (MRV) Heuristic
- Forward Checking

Each algorithm is executed step-by-step to demonstrate how solutions are derived.

### 🔍 Graph Traversal Visualizer
Supports visualization of:
- Breadth First Search (BFS)
- Depth First Search (DFS)

Users can observe queue and stack operations, visited nodes, and traversal sequences in real time.

### 🔢 Chromatic Number Finder
Determines the **minimum number of colors** required to color a graph without violating adjacency constraints.

### 🏁 Algorithm Derby
Provides side-by-side comparisons of CSP algorithms using metrics such as:
- Execution Time
- Nodes Explored
- Number of Backtracks

### 🎓 Narrated Teach Mode
Generates educational explanations describing why algorithms make specific decisions during execution.

### 🐍 Python Debugger Integration
Displays Python implementations of BFS and DFS while highlighting the currently executing lines of code.

### 🎮 Player vs AI Mode
Allows users to manually solve graph coloring problems and compare their solutions with AI-generated outputs.

### 📅 Examination Scheduler
Demonstrates a practical application of CSPs by generating conflict-free examination schedules using graph coloring principles.

## ⚙️ Installation and Setup

### Clone the Repository

```bash
git clone https://github.com/chiluverusrija/chromatic-ai.git
cd chromatic-ai
```

### Install Dependencies

```bash
npm install
```

### Start the Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```
